# To Do App 📝

Built this to learn Tkinter and how to save data with JSON.
Simple desktop app — add tasks, delete them, data saves automatically.
Added light/dark mode and priority colours after getting it working.

## What it does
- Add tasks by typing and clicking ADD or pressing Enter
- Delete selected task
- Mark task as complete with strike through effect
- Priority colour coding — High (red), Medium (orange), Low (green)
- Tasks saved to JSON so they persist after closing
- Light and dark mode toggle
- Shows task count at bottom

## What I learned
- Building desktop UI with Tkinter
- Saving and loading data with JSON
- Organising code into classes
- Key binding for Enter shortcut
- Using Canvas widget to fix button colours on macOS
- Building a theme system with dictionaries

## Tech used
Python, Tkinter, JSON

## How to run
```bash
python app.py
```
